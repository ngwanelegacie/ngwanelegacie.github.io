## Featured Projects

### Lending Club Loan Default Analysis
End-to-end credit risk analysis on **2.26M loans** ($50B+ in volume). Built a Gradient Boosting classifier that catches 66% of defaults before approval, translating to **~$95M in annual loss prevention** at portfolio scale. Includes SQL schema design, Python EDA, feature engineering, and an Excel executive summary.

**Stack:** Python (pandas, scikit-learn, matplotlib) · MySQL · Excel

[View Repo](https://github.com/ngwanelegacie/lending-credit-analysis) · [Live Site](https://ngwanelegacie.github.io)
